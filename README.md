# Smart Aquarium

This project is a simple IoT based smart Aquarium monitoring system using ESP32 and a local PHP/MySQL backend.

## Features
- Dashboard for reading temperature, status LED, status Heater and control feed servo, LED, etc.
- Record table page to see log data aquarium and can be export to xls file.
- Send data logging to local server.
- Automatic heater control based on threshold value, you can set in dashboard.

## Hardware Component
- ESP32
- DS18B20 temperature sensor
- LED
- Heater
- 2 Channel relays
- Motor Servo
